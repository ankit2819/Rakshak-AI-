Rakshak AI

Problem Statement

Last Minute Life Saver

Overview

Rakshak AI is an AI-powered emergency response platform designed to provide immediate assistance during critical situations. The system uses Google Gemini through Google AI Studio to analyze emergencies, generate intelligent guidance, and connect users with trusted contacts and nearby community responders.

Key Features

* Voice and SOS-based emergency activation
* AI-powered emergency classification using Gemini
* Real-time location sharing
* Emergency contact alerts
* Community Hero response network
* Medical ID (Blood Group, Allergies, Medications)
* AI-generated emergency guidance

Tech Stack

Frontend

* React Native
* Expo

Backend

* Node.js
* Express.js

Database

* Firebase Firestore

AI

* Google Gemini API
* Google AI Studio

Services

* Firebase Cloud Messaging (FCM)
* Google Maps API
* Twilio SMS

Project Goal

To reduce emergency response time by combining artificial intelligence, real-time communication, and a hyperlocal community support network that can provide immediate assistance during emergencies.
